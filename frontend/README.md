# AuxLess Frontend

AI-powered party music platform - eliminates aux cord battles using ML recommendations.

## Setup

1. Clone the repo and go to frontend folder:
git clone https://github.com/pvvishwesh-lang/AuxLess.git
cd AuxLess/frontend

2. Install dependencies:
npm install

3. Create `.env` file (copy from `.env.example`):
cp .env.example .env
Fill in your Firebase and YouTube credentials.

4. Run locally:
npm start

## Tech Stack
- React
- Firebase (Auth + Firestore)
- YouTube IFrame API
- Recharts

## Environment Variables
See `.env.example` for all required variables.
